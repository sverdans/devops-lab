# MAX_IN_ARR

Утилита, которая принимает массив целых чисел и возвращает максимальное значение

## Usage

```
max_in_arr <%d> [<%d> ...]
```

### example:

```sh
> max_in_arr 1 2 3 4 5 6 7
max: 7
```

## Build

requirements:
- gcc
- make
- libstdc++.so.6
- libgcc_s.so.1
- libc.so.6

```sh
make
make install
```
